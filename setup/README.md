# 🚀 Dev Environment Setup Scripts

Hey there! This is my personal collection of setup scripts that'll make my life easier when setting up a fresh dev environment. No more spending hours configuring everything manually - let's automate that boring stuff!

## 🎯 What's Inside?

- `mac.sh` - My carefully crafted setup script for macOS with the usual applications I use.

More platforms coming soon! 👀

## 🔧 How to Use

### For macOS
Fire up your terminal and run:
```bash
./mac.sh
```
Grab a coffee while the script does all the heavy lifting! ☕

**Note:** By default, you'll be prompted for confirmation before each application installation. To skip these prompts and install everything automatically, run:
```bash
./mac.sh -a
```
Keep in mind that some applications will still require your password to proceed with installation.
